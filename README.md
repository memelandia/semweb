# ⚡ SemWeb - ElectriPro

Sistema de Gestión para Electricistas. Aplicación web completa para gestionar presupuestos, conteo de planos, planificación de obras y registro de trabajos.

## Stack

- React 18 + Vite
- TailwindCSS 4
- Zustand (estado)
- Recharts (gráficos)
- Framer Motion (animaciones)
- jsPDF + html2canvas (exportar PDF)

## Desarrollo

```bash
npm install
npm run dev
```

## Deploy

Desplegado en Vercel automáticamente desde GitHub.
